# Proposal Solicitations

Master tracking list of funding calls. Each entry carries a **Status:** line
that `/status` reads to sort it into a bucket. The `vault-status` skill maps
these to the six buckets (Untouched → Idea Scoped → Drafting → Submitted →
Awarded → Complete).
