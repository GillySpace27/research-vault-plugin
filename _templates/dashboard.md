# Dashboard

Live task views, powered by the **Obsidian Tasks** plugin. Open this file in
Obsidian (with Tasks enabled) and the fenced ` ```tasks ` blocks below render
as interactive, aggregated task lists across every project file.

> Not using Obsidian? These blocks show as plain code — that's expected. Run
> `/dashboard` for a standalone `dashboard.html` board instead.

## Due or overdue

```tasks
not done
due before in 7 days
sort by due
group by filename
hide task count
```

## High priority

```tasks
not done
priority is above none
sort by priority
group by filename
hide task count
```

## No due date

```tasks
not done
no due date
sort by priority
group by filename
hide task count
```

## Inbox (unrouted)

```tasks
not done
path includes inbox
hide task count
```

## Everything open, by project

```tasks
not done
sort by due
group by filename
limit 200
```
